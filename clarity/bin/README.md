# Shell Scripts

Utility scripts for running and monitoring tests.

## Test Execution
- `run-fuzz-test.sh` - Run fuzz tests
- `check-test-status.sh` - Check test status

## Progress Monitoring
- `monitor-progress.sh` - Monitor test progress
- `monitor-fuzz-test.sh` - Monitor fuzz test progress
- `live-progress.sh` - Live progress display
- `show-progress.sh` - Show progress
- `show-progress-bar.sh` - Show progress bar
- `watch-progress.sh` - Watch progress
- `check-progress.sh` - Check progress

## Results Viewing
- `view-fuzz-results.sh` - View fuzz test results
- `check-fuzz-results.sh` - Check fuzz test results
- `check-fuzz-progress.sh` - Check fuzz test progress

## Status
- `status.sh` - General status check
